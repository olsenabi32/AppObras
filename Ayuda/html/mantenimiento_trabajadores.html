Gestión de Personal

Este módulo permite administrar el censo de operarios y técnicos de la empresa. Es fundamental para la asignación posterior en las obras.

Registro de Datos

  • Alta: Introduzca los datos personales. El sistema validará automáticamente el DNI.
  • Modificación: Seleccione un trabajador de la lista para editar su categoría o contacto.
Validación de DNI: El sistema no permitirá guardar si el DNI no cumple el formato (8 números + letra) o si el algoritmo de la letra es incorrecto.
Registros Duplicados: No se permite la entrada de dos trabajadores con el mismo identificador fiscal.
